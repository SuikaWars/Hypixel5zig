# Hypixel5zig-Beta
The 5zig mod plugin for the Hypixel Network.
## Features
### General
- Server
- Map(Arcade games not support)
### SkyWars
- Countdown
- Kills
- Assists
- Kit
- Win message
- Earned Coins
- Cage open countdown(Teams)
### SpeedUHC
- Countdown
- Kills
- Assists
- Kit
- Win message
- Earned Coins
- Cage open countdown(Teams)
