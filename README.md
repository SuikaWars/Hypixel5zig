# Hypixel5zig
## Select Language: [English](md/en_US.md) [日本語](md/ja_JP.md)
The 5zig mod plugin for the Hypixel Network.
## Features
### General
- Server
- Map(Arcade games not support)
### SkyWars
- Countdown
- Kills
- Assists
- Kit
- Win message
- Earned Coins
- Cage open countdown(Teams/Mega/Ranked)
### SpeedUHC
- Countdown
- Kills
- Assists
- Kit
- Win message
- Earned Coins
- Cage open countdown(Teams)
